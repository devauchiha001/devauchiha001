### 🤖 Smart Study Assistant — AI-Powered Full-Stack Web App

> *Architected, built, and shipped solo from zero to production.*

![React](https://img.shields.io/badge/React_18-20232A?style=flat&logo=react&logoColor=61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=flat&logo=typescript&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)
![Gemini](https://img.shields.io/badge/Google_Gemini_AI-4285F4?style=flat&logo=google&logoColor=white)
![Tailwind](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat&logo=tailwind-css&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat&logo=vite&logoColor=white)

**What it does:** A production-grade study platform that leverages Google Gemini AI to auto-generate MCQ quizzes and summarise lecture notes — including handwritten note images digitised by AI.

| Metric | Result |
|---|---|
| 📉 Study Prep Time Reduction | ~60% (user testing) |
| 🔐 Auth Methods | Email + OAuth (Google) |
| 🗄️ Database | Firestore with custom security rules |
| 📱 Platform | Mobile-first, PWA-ready |
| 🚀 Deployment | Firebase Hosting + CI/CD |

**Key Engineering Decisions:**
- Role-based access control via Firebase security rules — not just client-side guards
- Modular component architecture in React 18 for scalability and reusability
- PostCSS pipeline for optimised CSS output and Lighthouse score improvements
- AI pipeline handles both typed text and image input (handwritten notes via Gemini Vision)
